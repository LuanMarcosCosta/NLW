Esse material foi todo desenolvido pela rokeseat, mas o projeto sofrerá alterações como forma de estudo.
Utilizei as seguintes configurações (caso você queira rodar o projeto)

-- Node.js (baixe no site oficial)
-- git bash
-- git init -y (transforma em projeto os seus arquivos, projeto pra subir)
-- npm install express (vai instalar uma pasta com o nome "node_modules")

Depois digite:
-- npm start
-- depois abra o Chrome e digite "localhost3000"

Digite:
-- npm nunjucks


Caso você queira usar os pigins:
-- css formatter
-- Lauchbase Theme
-- Live Server
-- nunjucks Template
-- vs code icons

Nota: Caso você tenha configurado tudo e deu algum erro, ou não mudou quase nada, copie e cole:
(dentro de settings.json - CTRL + SHIFT + P)

"files.associations": {
        "*.html": "njk"
    },
    "emmet.includeLanguages": {
        "njk": "html"
    },

    "vsicons.associations.files": [
        { "icon": "nunjucks", "extensions": ["html"], "format": "svg" }
    ],
    "workbench.iconTheme": "vscode-icons"

}
